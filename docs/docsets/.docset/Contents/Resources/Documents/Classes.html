  • BMLT_MeetingSearch_AnimatedButtonView

    This is a special class that presents a button. When pressed, it starts animating images with up to 100 frames.

    It can be initialized in Interface Builder with a frame prefix (we add a no-leading-zero integer between 0 and 99), and an initial index.

Prefs Class
  • BMLT_MeetingSearch_Prefs

    This is a very simple persistent user prefs class. It is instantiated as a SINGLETON, and provides a simple, property-oriented gateway to the simple persistent user prefs in iOS. It shouldn’t be used for really big, important prefs, but is ideal for the basic settings type of prefs most users set in their gear screen.
